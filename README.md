# DevConnect 🧑‍💻👩‍💻

> **A social networking platform exclusively designed for developers to connect, match, and collaborate.**

DevConnect is a full-stack web application built with the **MERN stack** (MongoDB, Express, React, Node.js). It provides a familiar "swipe-to-match" interface tailored specifically for the software engineering community, allowing developers to find mentors, mentees, or collaborators for their next big project.

---

## ✨ Features

*   **Secure User Authentication:** Sign up and log in securely utilizing JSON Web Tokens (JWT) and HTTP-only cookies.
*   **Developer Profiles:** Create and customize your developer profile with your skills, bio, and avatar.
*   **Interactive Feed:** Browse through other developers' profiles in a dynamic feed.
*   **Connection System:** Send connection requests (swipe right/interested) or pass (swipe left/ignore).
*   **Manage Connections:** View received requests, accept or reject them, and browse your list of established connections.
*   **Responsive UI:** A beautiful, mobile-friendly interface built with Tailwind CSS and DaisyUI.
*   **Global State Management:** Efficient state handling using Redux Toolkit.
